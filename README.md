# TRANQUIDESCANSO S.A. – Sistema de Reservas y Gestión Hotelera

Aplicación académica para el diseño e implementación de una base de datos relacional y una aplicación conectada para la cadena de hoteles TRANQUIDESCANSO S.A., enfocada en la gestión de reservas, huéspedes, habitaciones, categorías de hotel y pagos asociados a la estadía y servicios adicionales.[web:1][web:8]

## Características principales

- Gestión de hoteles (datos generales, categoría, historial de cambios de categoría).
- Gestión de tipos y unidades de habitación por hotel.
- Registro de huéspedes y agencias de viaje.
- Gestión completa de reservas:
  - Reservas directas de huéspedes o a través de agencia.
  - Registro de fechas (creación, inicio, fin), número de personas, número y tipo de habitaciones.
  - Servicios adicionales (parqueadero, mascotas, etc.).[web:3][web:12]
- Manejo de pagos de reserva:
  - Anticipo del 20% antes de 24 horas.
  - Cancelación automática si no se paga el anticipo.
  - Pago del 80% restante en el registro de llegada (check‑in).
- Control de check‑in/check‑out:
  - Ventana de registro de entrada (3:00 pm a 7:00 pm).
  - Asignación de habitaciones a cada huésped con identificación individual.
  - Registro y cobro de servicios adicionales hasta el check‑out.[web:9][web:15]
- Consultas soportadas (entre otras):
  - Reservas en un período de tiempo.
  - Reservas canceladas sin anticipo.
  - Reservas no utilizadas con anticipo pagado.
  - Reservas con llegada a tiempo vs. tardía/no presentada.
  - Reservas con menores de edad y/o mascotas.
  - Reservas que pagaron servicios adicionales.
  - Datos de huéspedes de una reserva específica.[web:2][web:19]
